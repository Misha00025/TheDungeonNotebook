The Dungeon Notebook API

Документация API

The Dungeon Notebook — это веб-приложение для управления настольными ролевыми играми (D&D, Pathfinder и др.). API предоставляет полный набор endpoint'ов для работы с группами, персонажами, предметами, навыками, заметками и схемами.
