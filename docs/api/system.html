⚙️ Системные

Служебные endpoint'ы для получения информации о состоянии API.

GET /get_api no auth
⚙️ handler: get_api

Возвращает схему всех зарегистрированных API-методов.

Ответ

{
  "api_methods": "Array<object>"
}
200 OK
GET /whoami auth

Информация о текущем пользователе или группе.

Ответ

{
  "id": "int | null",
  "type": "'user' | 'group' | 'anonymous'"
}
200 OK
