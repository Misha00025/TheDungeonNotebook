🔐 Аутентификация

Регистрация, вход и обновление JWT-токенов.

POST /auth/register no auth

Регистрация нового пользователя.

Тело запроса

{
  "username": "string",
  "password": "string"
}

Ответ

{
  "id": "int"
}
201 Created 409 Conflict
POST /auth/login no auth

Вход в систему.

Тело запроса

{
  "username": "string",
  "password": "string"
}

Ответ

{
  "token": "<JWT>"
}
200 OK 401 Unauthorized
POST /auth/refresh no auth
⚙️ handler: auth_refresh

Обновление JWT-токена. Требуется заголовок Refresh-Token.

Ответ

{
  "accessToken": "<JWT>"
}
200 OK 401 Unauthorized
