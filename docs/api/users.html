👤 Пользователи

Управление профилями пользователей.

GET /users no auth

Список пользователей.

ПараметрТипОписание
idsstringID через запятую (1,2,3)
nicknamestringФильтр по нику

Ответ

{
  "users": ["Array<User>"]
}
200 OK
POST /users auth
⚙️ handler: user_create

Создание профиля пользователя. ID подставляется из JWT.

Тело запроса

{
  "nickname": "string",
  "visibleName"?: "string",
  "imageLink"?: "string"
}

Ответ

{
  "id": "int",
  "nickname": "string",
  "visibleName": "string",
  "imageLink": "string"
}
201 Created 400 Bad Request 409 Conflict
GET /users/{id} auth

Получение профиля пользователя.

Ответ

{
  "id": "int",
  "nickname": "string",
  "visibleName": "string",
  "imageLink": "string"
}
200 OK 404 Not Found
PATCH /users/{id} auth self_only

Обновление профиля (только владелец).

Тело запроса

{
  "visibleName"?: "string",
  "imageLink"?: "string"
}

Ответ

{
  "id": "int",
  "nickname": "string",
  "visibleName": "string",
  "imageLink": "string"
}
200 OK 400 Bad Request 404 Not Found
