📁 Группы

Управление группами (кампаниями) и их участниками.

GET /groups auth
⚙️ wrap: groups

Список групп пользователя.

Ответ

{
  "groups": [{"id": "int","name": "string","icon": "string | null"}]
}
200 OK
POST /groups auth

Создание группы.

Тело запроса

{
  "name": "string",
  "icon"?: "string"
}

Ответ

{
  "id": "int"
}
201 Created
GET /groups/{id} auth

Получение группы.

Ответ

{
  "id": "int",
  "name": "string",
  "icon": "string | null"
}
200 OK 404 Not Found
PATCH /groups/{id} auth group_admin

Обновление группы.

Тело запроса

{
  "name"?: "string",
  "icon"?: "string"
}

Ответ

null
200 OK 400 Bad Request 404 Not Found
GET /groups/{id}/users auth
⚙️ handler: group_users

Участники группы (композитный: policy + users).

Ответ

{
  "users": [{"user": {"id": "int","nickname": "string","visibleName": "string","imageLink": "string"},"isAdmin": "bool"}]
}
200 OK
PUT /groups/{id}/users/{userId} auth group_admin

Добавить участника в группу.

Тело запроса

{
  "groupId": "int (из path)",
  "userId": "int (из path)"
}

Ответ

null
201 Created 409 Conflict
DELETE /groups/{id}/users/{userId} auth group_admin

Удалить участника из группы.

ПараметрТипОписание
groupIdintID группы
userIdintID пользователя

Ответ

null
200 OK
