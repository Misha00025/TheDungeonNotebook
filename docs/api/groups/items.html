⚔️ Предметы группы

Управление предметами в группе.

GET /groups/{id}/items auth

Список предметов группы.

Ответ

{
  "items": [{"id": "int","name": "string","description": "string","attributes": "Array","price": "int","amount"?: "int","isSecret": "bool"}]
}
200 OK
POST /groups/{id}/items auth group_admin

Создать предмет группы.

Тело запроса

{
  "name": "string",
  "description": "string",
  "price"?: "int",
  "attributes"?: "Array",
  "isSecret"?: "bool",
  "amount"?: "int"
}

Ответ

null
201 Created
